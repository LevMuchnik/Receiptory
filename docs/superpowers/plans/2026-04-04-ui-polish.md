> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Restyle the entire Receiptory frontend with a consistent Slate + Blue, dense data-driven aesthetic that works across desktop/mobile and light/dark modes.

**Architecture:** Replace the Material Design-inspired CSS custom properties with a Slate + Blue palette. Update Tailwind class names across all components for tighter spacing, border-based elevation, and consistent typography (Inter only). No structural or logic changes.

**Tech Stack:** React, TypeScript, Tailwind CSS v4, shadcn/ui (base-ui variant)

**Visual verification:** After each task, run `cd frontend && npm run build` to verify no compile errors. For visual checks, run `cd frontend && npm run dev` and inspect in browser.
